## Mobile Experience

### Option 1: Command Palette
- Top nav collapses to hamburger
- Command palette works on mobile
- User menu accessible

### Option 2: Workspace-Based
- Sidebar becomes bottom nav on mobile
- User area stays visible
- Recent items accessible

### Option 3: Hub & Spoke
- Cards stack vertically
- Contextual sidebar becomes drawer
